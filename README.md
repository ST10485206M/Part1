# Cybersecurity Awareness Bot

## Description
This project is a C# console application designed to promote cybersecurity awareness. 
The program interacts with users through a menu system and provides useful tips on staying safe online. 
It also includes a chatbot feature and uses speech synthesis to greet the user.

## Features
- Voice greeting using SpeechSynthesizer
- Interactive menu system
- Cybersecurity tips:
  - Password Safety
  - Phishing Awareness
  - Safe Browsing
- Chatbot mode for user interaction
- Typing animation effect
- ASCII logo display

## How to Run
1. Download the project from GitHub
2. Open the file `Part1POE.sln` in Visual Studio
3. Press `F5` or click the Start  button
4. Follow the on-screen instructions

## Technologies Used
- C#
- .NET Framework
- System.Speech.Synthesis
- Visual Studio

## Program Overview
The application starts with a voice welcome message and displays a logo.  
Users enter their name and are presented with a menu to choose different cybersecurity topics.  
The chatbot feature allows basic interaction and responses based on user input.
